# Zettelkasten

Bem-vindo ao meu **Zettelkasten** — um sistema de notas curtas e interconectadas inspirado no método de [Niklas Luhmann](https://pt.wikipedia.org/wiki/Niklas_Luhmann).

## Como navegar

- Use a **sidebar** à esquerda para explorar as notas disponíveis
- Siga os **links entre notas** para descobrir conexões
- Use as **tags** para encontrar notas por tema
- Explore o **mapa de conexões** abaixo para visualizar a rede
