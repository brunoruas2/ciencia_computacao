# Agent.md - Resumo do Projeto

## Visao Geral

Site pessoal de estudos em Ciencia da Computacao, construido com **Docusaurus v3.7.0** e hospedado no **GitHub Pages**. O projeto funciona como um "segundo cerebro" de aprendizado, reunindo anotacoes de cursos, faculdade e estudos autodidatas.

- **URL:** https://brunoruas2.github.io/CC_site/
- **Autor:** Bruno de Melo Ruas
- **Idioma:** Portugues (pt)
- **Licenca:** MIT

## Stack Tecnica

| Componente        | Tecnologia                          |
| ----------------- | ----------------------------------- |
| Framework         | Docusaurus 3.7.0                    |
| Linguagem         | TypeScript / MDX                    |
| UI                | React 18                            |
| Hospedagem        | GitHub Pages                        |
| Busca             | docusaurus-lunr-search              |
| Diagramas         | Mermaid (@docusaurus/theme-mermaid) |
| Formulas          | KaTeX (remark-math + rehype-katex)  |
| Icones            | react-icons                         |
| Timeline          | react-vertical-timeline-component   |
| Syntax Highlight  | prism-react-renderer                |

```
├── src/

```

## Componentes Customizados

- **HomepageFeatures** (`src/components/HomepageFeatures/`) - Cards na homepage com aviso, sobre o autor e links
- **backlogTable.mdx** (`docs/_src/`) - Tabela de progresso do backlog
- **introCard.mdx** (`docs/_src/`) - Card de introducao reutilizavel
- **Timeline de carreira** - Na intro, usando react-vertical-timeline-component

## Scripts Disponiveis

```bash
npm start          # Dev server local
npm run build      # Build de producao
npm run deploy     # Deploy para GitHub Pages
npm run serve      # Servir build local (necessario para busca)
npm run typecheck  # Verificacao de tipos TypeScript
npm run clear      # Limpar cache
```

## Configuracoes Importantes

- **Sidebar:** Auto-gerada a partir da estrutura de pastas (`autogenerated`)
- **Tema:** Verde (#2e8555) em light mode, ciano (#25c2a0) em dark mode
- **Deploy:** `gh-pages` branch via `docusaurus deploy`
- **Node:** Requer >= 18.0

## Convencoes do Projeto

- Arquivos numerados com prefixo `XX-` para ordenacao (ex: `01-intro.md`)
- Conteudo em portugues
- Formato MDX para paginas que precisam de componentes React
- Formato MD para conteudo puramente textual
- Imagens armazenadas em subdiretorios `img/` dentro de cada secao
